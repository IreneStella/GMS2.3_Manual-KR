array_set

이 함수를 사용하면 배열 인덱스 안에 있는 값을 설정해 줄수 있습니다. 함수의 인자값으로 배열을 가지고 있는 변수, 값을 설정해야 할 인덱스 그리고 설정하고 싶은 값 입니다. 이 함수는 다음 패턴을 바탕으로 특정 배열의 차원을 인자값으로 넣어주게 되면 다차원(multi-dimension)배열 또한 값을 설정할 수 있습니다.

// 1차원 배열
array_set(my_array, 0, 100);
// 2차원 배열
array_set(my_array[0], 0, 100);
// 3차원 배열
array_set(my_array[0][0], 0, 100);
// 기타 등등...

 

문법:

array_set(variable, index, value);

인자값(Argument) 설명(Description)
variable 배열을 가지고 있는 변수.
index 값을 설정하고자 하는 배열의 인덱스.
value 설정하고 싶은 값.

 

반환값:

N/A

 

예제:

for (var i = 0; i < 10; ++i;)
    {
    array_set(score_array, i, i*100));
    }

위 코드는 인자 값으로 넣어준 배열의 처음 10개의 항목의 값을 for루프를 사용해서 설정해줍니다.
