array_create

이 함수를 사용하시면 1차원 사이즈의 배열을 생성할 수 있습니다. 함수 인자값으로 생성하고픈 배열의 길이를 넣어주시면, 함수는 변수를 할당 할 수 있는 배열을 반환하게 될 것입니다. 이 함수를 사용해서 생성된 배열의 각 항목들은 (추가적으로) 초기값을 정해주지 않다면 0으로 초기화할 것입니다. 만약 함수의 인자값으로 배열의 초기값을 넣어주게 된다면, 모든 항목들의 값은 0이 아닌 넣어주신 인자값으로 초기화가 되겠지만, 이렇게 초기화를 할 경우 성능 오버헤드(오버헤드(overhead)는 어떤 처리를 하기 위해 들어가는 간접적인 처리 시간 · 메모리 등을 말한다.) 늘어나게 될 것입니다.

 

문법:

array_create(size, [value]);

인자값(Argument) 설명(Description)
size 생성하고자 하는 배열의 크기
value 배열의 모든 요소(indices)를 초기화 할 값(설정하지 않으면 0).

 

반환값:

Array Reference

 

예제:

instance_array = array_create(100, noone);

위 코드는 크기 100을 가진 새 배열을 생성합니다. 각 항목은 함수를 사용해서 noone 키워드로 초기화하고, "instance_array"란 변수에 이 값을 넣어줍니다.
