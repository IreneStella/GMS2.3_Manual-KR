variable_struct_set

이 함수를 사용하면 구조체 참조자안에 변수값을 설정할 수 있습니다. 함수의 인자값으론 설정하고 싶은 변수의 이름(문자형 데이터) 뿐만 아니라 구조체 참조자도 들어가야 하고, 마지막으로 변수에 설정하고 싶은 값(data type 항목에 부합되는 어떠한 형태든 상관없다)을 넣어줘야 합니다(아래 예제코드에서 확인가능). 만약 변수가 구조체 참조자 내부에 존재하지 않다면, 인자로 넣어준 변수이름과 값으로 생성하고, 초기화할 것입니다.

 

문법:

variable_struct_set(struct, name, val);

인자값(Argument) 설명(Description)
struct 설정하고 싶은 구조체 참조자
name 설정하고 싶은 변수의 이름(문자형 데이터)
val 변수에 설정하고 싶은 값

 

반환값:

N/A

 

예제:

if !variable_struct_exists(mystruct, "shields")
   {
   variable_struct_set(mystruct, "shields", 0);
   }

위 코드는 주어진 구조체 참조자안에 변수의 존재여부를 확인하고, 해당 이름의 변수가 존재하지 않다면 새로 생성하고 0으로 초기화 해줍니다.
