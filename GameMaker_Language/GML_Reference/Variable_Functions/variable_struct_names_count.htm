variable_struct_names_count

이 함수를 사용하면 구조체안에 정의된 전체 변수의 수를 알 수 있습니다. 변수의 인자값으로 확인하고 싶은 구조체 ID(the struct ID)를 넣어주면 변수의 수를 센 정수 값을 반환할 것이고, 인자로 넣은 구조체 ID가 존재하지 않다면 -1을 반환할 것입니다.

 

문법:

variable_struct_names_count(struct_id);

인자값(Argument) 설명(Description)
struct_id 확인하고 싶은 특정 구조체 id값

 

반환값:

Integer

 

Example:

var _num = variable_struct_names_count(mystruct);
show_debug_message("Struct Variables = " + string(_num));

위 코드는 인자값으로 넣어준 구조체의 변수의 숫자를 검색하고, 콘솔 출력창에 디버그 메세지로 해당 값을 출력할 것입니다.
