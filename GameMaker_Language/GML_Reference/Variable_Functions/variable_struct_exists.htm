variable_struct_exists

이 함수를 사용하시면 구조체 참조자(struct reference) 내부에 변수가 존재 여부를 확인할 수 있습니다. 함수의 인자값으로 체크하실 변수의 이름(문자형 데이터)뿐만 아니라 구조체 참조자도 넣으셔야 합니다(아래 예제코드에서 확인가능). 함수는 구조체 참조자 내부에 변수가 존재한다면 true를 반환할 것이고, 그게 아니라면 false를 반환합니다.

 

문법:

variable_struct_exists(struct, name);

인자값(Argument) 설명(Description)
struct 확인하고 싶은 구조체 참조자
name 확인하고 싶은 구조체 참조자 변수 이름(문자형 데이터)

 

반환값:

Boolean

 

예제:

if !variable_struct_exists(mystruct, "shields"))
{
mystruct.shields = 0;
}

위 코드는 "shields"이름을 가진 변수가 인자값으로 넣어준 구조체 참조자안에 존재하는지 확인합니다. 만약 존재하지 않는다면 해당 이름을 가진 변수를 생성하고 0으로 초기화 합니다.
