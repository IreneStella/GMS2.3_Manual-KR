instance_of

이 함수는 constructor function(생성자 함수) 또는 new 연산자를 사용해서 구조체를 생성할 때 쓰인 함수의 이름을 찾는데 사용됩니다. 함수인자값으로 확인하고 싶은 구조체 참조자를 넣어주면, 함수를 사용했다면 문자열 데이터로 함수의 이름을 반환하고, 그렇지 않다면 "undefined"를 반환합니다(구조체 참조자 이외에 변수를 넣어주면 undefined가 출력됨). 만약 구조체 리터럴(a struct literal : 구조체를 생성할 때 "생성자 함수"를 사용하지 않는 참조자를 의미함)인 참조자를 인자값으로 넣어주면 함수는 단순히 "struct"이란 문자열 데이터값을 반환할 것입니다.

 

문법:

instanceof(struct_id);

인자값(Argument) 설명(Description)
struct 생성자 함수를 알고 싶은 구조체 참조자

 

반환값:

String or undefined

 

확장 예제:

이 예제에선 우리는 먼저 구조체 생성자 함수로 사용될 함수를 정의를 해야 합니다. :

function init_struct(_a, _b, _c) constructor
    {
    a = _a;
    b = _b;
    c = _c;
    }

정의된 함수는 new연산자를 같이 사용해서 구조체를 생성하면서 인자값으로 넣어준 값으로 함수 내부에 선언된 변수들을 초기화 해줍니다. :

mystruct = new init_struct(10, 100, "Hello World");

그런다면 함수의 이름을 다음과 같은 방법으로 얻으실 수 있습니다. :

var _name = instanceof(mystruct);
if is_string(_name)
    {
    show_debug_message(_name);
    }
