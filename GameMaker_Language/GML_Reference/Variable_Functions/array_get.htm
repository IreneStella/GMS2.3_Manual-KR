array_get

이 함수를 사용하시면 배열의 인덱스(index)에서 값을 검색할 수 있습니다. 이 함수의 인자값으로 배열을 가지고 있는 변수와 값을 얻고자 하는 인덱스가 필요합니다. 만약 인자값으로 넣어준 인덱스가 배열의 범위를 넘어가게 된다면 게임이 심각한 에러로 망가질 수 있으니 이점을 숙지하셔야 합니다. 이 함수는 다음 패턴을 바탕으로 배열을 가진 변수를 인자값으로 넣어주게 되면 다차원(multi-dimension)배열 또한 사용할 수 있습니다.

// 1차원 배열
array_get(my_array, 0);
// 2차원 배열
array_get(my_array[0], 0);
// 3차원 배열
array_get(my_array[0][0], 0);
// 기타 등등...

 

문법:

array_get(variable, index);

인자값(Argument) 설명(Description)
variable 배열을 가지고 있는 변수.
index 가져올 값이 있는 배열의 인덱스.

 

반환값:

Any valid data type that an array can hold.

 

예제:

for (var i = 0; i < 10; ++i;)
    {
    show_debug_message(array_get(my_array, i));
    }

위 코드는 "my_array"배열의 처음 10개 항목(item)을 콘솔창에 출력을 할 것입니다.
