array_equals

이 함수를 사용하면 두 배열이 동일한지 확인할수 있습니다. 함수의 인자값으로 검사할 두 배열을 넣어주고, 두 배열의 항목들의 값이 같다면(the two arrays hold equivalent values) true을 반환할 것이고, 같지 않다면 false를 반환할 것입니다. 이 함수로 두 배열을 비교하는 것은 "=="를 사용해서 비교하는 것과 다릅니다. "=="을 사용해서 두 배열을 비교하는 것은 단순히 배열의 참조자(reference)를 비교하는 것이고, 이것은 배열의 이름을 비교하는 것입니다. 예를 들어서:

var a = [1,2,3,4];
var b = [1,2,3,4];
if (a == b)
   {
   show_debug_message( "이 조건문은 a와 b 참조자(reference)들이 글자가 다르므로 절대 실행되지 않습니다." );
   }
if (array_equals(a, b))
   {
   show_debug_message( "함수로 비교한 배열은 true와 false를 반환하기에 비교문이 동작할 것입니다." );
   }

 

문법:

array_equals(var1, var2);

인자값(Argument) 설명(Description)
var1 The index of the first array.
var2 The index of the second array.

 

반환값:

Boolean

 

예제:

if !array_equals(inventory_array, item_array)
   {
   var len = array_length(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

위 코드를 보면 두 배열이 동일한 값을 가졌는지 체크하고, 동일하지 않다면 "inventory_array"의 배열 통쨰로 복사해서 "item_array"배열에 붙여넣기를 합니다.
