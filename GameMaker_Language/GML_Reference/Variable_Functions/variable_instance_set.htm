variable_instance_set

이 함수를 사용하면 개체에서 주어진 변수의 값을 설정할 수 있습니다. 문자열로써 체크되는 변수이름뿐만 아니라 고유한 개체 ID(Instance ID)값(룸 에디터의 Instance Properties 에서 찾을 수 있거나, 함수 instance_create_layer() 를 호출할 때 반환되는 값)을 인자값으로 받고, 마지막으로 변수의 값을 설정하고 싶은 값( data type 항목에 부합되면 어떤 형태든 상관없다) 을 넣어주면 됩니다(샘플 코드는 아래에서 확인할 수 있다). 만약 개체안에 존재하지 않는 변수이름을 함수의 인자로 넣게 된다면 해당 변수이름의 값을 생성하고 값을 부여하게 됩니다.

 

문법:

variable_instance_set(instance_id, name, val);

인자값(Argument) 설명(Description)
instance_id 사용할 개체의 고유의 ID값
name 설정하고 싶은 변수의 이름(문자열 데이터)
val 변수에 설정하고 싶은 값

 

반환값:

N/A

 

예제:

if !variable_instance_exists(id, "shields")
   {
   variable_instance_set(id, "shields", 0);
   }

위 코드를 보면 먼저 개체안에 해당 변수이름이 존재하지 않는 것을 체크하고, 존재하지 않는 경우 해당 변수이름으로 새로운 변수를 생성하고 값을 0으로 설정합니다.
