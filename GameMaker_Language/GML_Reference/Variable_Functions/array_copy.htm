array_copy

이 함수를 사용하시면 배열의 전체 또는 일부분을 다른위치에 있는 또 다른 배열에 복사할 수 있습니다. 함수의 인자 값으로 두 배열의 정보를 넣어야 하는데, 하나는 복사하고 싶은 배열(source)과 다른 하나는 값을 넣어줄 배열(destination)을 넣어줘야 합니다(두 배열은 함수를 사용하기전에 생성되어 있어야 합니다). 또한 두 배열 뿐만아니라 시작 배열(source)의 복사하고자 하는 위치와 끝 배열(destination)의 붙여넣기할 위치도 넣어줘야합니다. 마지막 인자값으로 복사할 배열의 길이(또는 길이의 일부분)을 지정해줘야 합니다. 만약 시작 배열(source)에서 복사한 데이터가 끝 배열(destination)의 길이를 초과하게 된다면, 끝 배열의 길이는 데이터를 수용하기 위해서 확장되게 됩니다.

이 함수는 다음 패턴을 바탕으로 배열의 인덱스(index)를 인자값으로 넣어주게 되면 다차원(multi-dimension)배열 또한 사용할 수 있습니다.

// 배열의 첫번째(first) 차원 배열에서
// 두번째(second) 차원 배열로 복사
array_copy(item_array, 0, inventory_array[0], 0, len);

// 첫번째(first) 배열에서
// 세번째(third) 차원배열로 복사
array_copy(item_array[0][0], 0, inventory_array, 0, len);

//기타 등등...

 

 

문법:

array_copy(dest, dest_index, src, src_index, length);

인자값(Argument) 설명(Description)
dest 붙여넣기하고 싶은 배열의 id.
dest_index 붙여넣기할 배열의 항목(index)위치.
src 복사할 배열의 id.
src_index 복사할 배열의 항목(index)에서 복사시작위치.
length 복사할 길이(배열의 길이).

 

반환값:

N/A

 

예제:

if !array_equals(inventory_array, item_array)
   {
   var len = array_length(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

위 코드는 두 배열이 동일한 값을 보유하는지 확인하고, 만약 동일한 값이 없다면 "inventory_array"배열의 전체 요소들을 복사해서 "item_array"로 붙여넣기 합니다.
