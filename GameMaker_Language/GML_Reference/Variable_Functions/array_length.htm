array_length

이 함수를 사용하시면 어떤 배열차원(an array dimension)의 길이(전체 길이)를 얻으실 수 있습니다. 함수 인자값으로 배열의 참조자 (역자 : 원본에선 배열의 index라고 표시하는데 함수의 동작으로 보면 reference를 넣는 것으로 보고 참조자로 번역.) 를 넣어주면, 함수는 해당 참조자가 가리키는 배열의 전체 길이를 정수값으로 반환할 것이니다. 이 함수는 다음 패턴을 바탕으로 배열 참조자를 인자값으로 넣어주게 되면 다차원(multi-dimension)배열 또한 사용할 수 있습니다.

// 1차원 배열의 반환
val = array_length(my_array);

// 2차원 배열의 반환
val = array_length(my_array[0]);

// 3차원 배열의 반환
val = array_length(my_array[0][0]);

// 기타 등등...

 

문법:

array_length(array_index);

인자값(Argument) 설명(Description)
array_index 길이를 검사하고 싶은 배열의 참조자

 

반환값:

Integer

 

예제:

for (var i = 0; i < array_length(a); ++i;)
    {
    a[i] = -1;
    }

위 코드는 배열 속을 루프를 사용해서 전체 항목을 -1으로 설정해주는 것입니다.
