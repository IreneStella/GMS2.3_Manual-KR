variable_struct_get_names

이 함수를 사용하시면 구조체 속 변수의 이름으로 구성된 배열을 받을 수 있습니다. 함수의 인자값에 확인하고 싶은 구조체 참조자를 넣어주면, 반환되는 배열의 각 항목들은 구조체 속에 있는 변수의 이름(문자형 데이터)으로 나타낼 것 있습니다.

 

문법:

variable_struct_get_names(struct);

인자값(Argument) 설명(Description)
struct 확인하고 싶은 구조체 참조자

 

반환값:

Array (each entry is a string)

 

Example:

var str = "";
var array = variable_struct_get_names(mystruct);
show_debug_message("Variables for struct: " + string(array));
for (var i = 0; i < array_length(array); i++;)
    {
    str = array[i] + ":" + string(variable_struct_get(mystruct, array[i]));
    show_debug_message(str);
    }

위 코드는 인자로 넣어준 구조체의 변수이름으로 구성된 배열을 받고, 그것을 디버그 출력에 변수이름이랑 대응하는 값을 출력합니다.
