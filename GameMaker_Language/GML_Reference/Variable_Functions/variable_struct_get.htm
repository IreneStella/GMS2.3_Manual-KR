variable_struct_get

이 함수를 사용하시면 구조체 참조자안에 있는 변수이름의 값을 가져올 수 있습니다. 함수의 인자값으론 얻고자 하는 변수의 이름(문자형 데이터) 뿐만 아니라 변수가 들어있는 구조체 참조자를 넣으셔야 합니다.(아래 예제코드에서 확인 가능). 함수는 인자로 넣어준 변수이름의 값을 반환하거나, 구조체 참조자안에 변수가 없다면 undefined 키워드를 반환할 것입니다.

주의! 만약 인자로 넣어준 변수가 구조체 참조자안에 존재하지 않는다면 함수는 "undefined" 키워드를 반환할 것 입니다. 그리고 이것은 당신의 게임에서 에러를 일으키고 함수가 동작할 때 게임이 멈춰버릴 수 있습니다. 혹시 이런 증상이 의심이 된다면 먼저variable_struct_exists() 함수를 사용해서 인자로 넣은 변수가 존재하는지 확인하시길 바랍니다.

 

문법:

variable_instance_get(struct_id, name);

인자값(Argument) 설명(Description)
struct 확인하고 싶은 구조체 참조자
name 값을 얻고자 하는 변수의 이름(문자형 데이터)

 

반환값:

Value (any data type) or undefined

 

Example:

if variable_struct_exists(mystruct, "shields")
    {
    var ss = variable_struct_get(mystruct, "shields");
    }
else
    }
    var ss = -1;
    }

위 코드는 인자값으로 넣어준 구조체 참조자에 변수이름의 존재를 확인하고, 변수가 존재한다면 변수의 값을 지역변수에 저장합니다. 존재하지 않는다면 지역변수의 값을 -1로 설정합니다.
