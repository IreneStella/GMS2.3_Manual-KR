array_resize

이 함수를 사용하시면 기존 배열 차원의 길이를 새로운 길이로 재조정할(resize) 수 있습니다. 함수의 인자값으로 배열의 인덱스들(indices) 길이와 재조정할 배열을 넣어주면 함수가 배열을 재조정할 것입니다. 이 함수는 배열의 사이즈를 줄이는 용도로 설계 되었다는 것을 아셔야합니다. 배열의 길이는 늘리는 것은 단순히 새 인덱스를 선언해서 넣어주면 되는 간단한 작업입니다. 배열의 길이를 늘리기 되면 새로 생성된 배열은 0으로 디폴트(default)값이 되어 설정됩니다.

이 함수는 다음 패턴을 바탕으로 특정 배열의 차원을 인자값으로 넣어주게 되면 다차원(multi-dimension)배열 또한 길이를 재조정할 수 있습니다.

// 1차원배열 길이 조정(resize)
array_resize(my_array, 10);

// 2차원배열 길이 조정(resize)
array_resize(my_array[0], 10);

// 3차원배열 길이 조정(resize)
array_resize(my_array[0][0], 10);

// 기타등등...

 

문법:

array_resize(array_index, new_size);

인자값(Argument) 설명(Description)
array_index 길이를 재조정하고 싶은 배열
new_size 배열의 새로 정해줄 길이 (정수값, 0 부터 넣을 수 있음).

 

반환값:

N/A

 

예제:

if array_length(a) > 10
    {
    array_resize(a, 10);
    }

위 코드는 배열의 길이를 체크하고, 만약 길이가 10보다 넘으면 10으로 길이를 재조정합니다.
