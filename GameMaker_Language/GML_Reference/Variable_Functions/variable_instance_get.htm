variable_instance_get

이 함수를 사용하면 지명된 변수에서 값을 얻을 수 있습니다. 문자열로써 체크되는 변수이름뿐만 아니라 고유한 개체 ID(Insance ID)값(룸 데이터의 Instance Properties 에서 찾을 수 있거나, 함수 instance_create_layer() 를 호출할 때 반환되는 값)을 인자값으로 넣어줘야 합니다(샘플 코드는 아래에서 확인할 수 있다). 이 함수는 인자값으로 넣어준 "변수이름"의 값을 반환하고, 변수가 존재하지 않으면 "undefined"라고 출력됩니다.

주위! 만약 존재하지 않는 변수의 값을 함수의 인자로 넣게 된다면 반환값으로 undefined 키워드를 출력할 것입니다. 그리고 이것은 당신의 게임에서 에러를 일으키고 함수가 동작할 때 게임이 멈춰버릴 수 있습니다. 혹시 이런 증상이 의심이 된다면 먼저 variable_instance_exists 함수를 사용해서 인자로 넣은 변수가 존재하는지 확인하시길 바랍니다.

 

문법:

variable_instance_get(instance_id, name);

인자값(Argument) 설명(Description)
instance_id 사용할 개체의 고유한 ID값
name 값을 얻고자 하는 변수의 이름(문자형 데이터)

 

반환값:

Value (any data type) or undefined (if the named variable does not exist)

 

예제:

if variable_instance_exists(id, "shields")
   {
   var ss = variable_instance_get(id, "shields");
   }
else
   }
   var ss = -1;
   }

위 코드는 변수가 존재한다면 변수가 가진 값을 지역변수로 저장할 것이고, 만약 존재하지 않는다면 지역변수로 저장되는 값은 -1로 설정하게 됩니다.
