variable_instance_names_count

이 함수를 사용하면 개체에 정의된 총 변수값의 수를 알아낼 수 있습니다. 함수의 인자값에 확인할 개체 ID를 넣으면, 함수는 개체내에 변수의 수를 정수값으로 반환해주고, 또는 -1(개체에 지정된 개체ID값이 없을 경우)을 반환해 줄 것입니다.

 

문법:

variable_instance_names_count(struct_id);

인자값(Argument) 설명(Description)
instance_id 확인할 특정 개체의 ID 값.

 

반환값:

Integer

 

예제:

var _num = variable_instance_names_count(mystruct);
show_debug_message("Instance Variables = " + string(_num));

위 코드는 주어진 개체에 있는 변수의 수를 검색해주고 콘솔창에 디버그 메세지로 변수 숫자를 출력을 해줍니다.
