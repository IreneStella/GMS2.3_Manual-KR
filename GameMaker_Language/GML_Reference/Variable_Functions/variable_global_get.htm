variable_global_get

이 함수를 사용하시면 인자값으로 넣어준 전역변수의 값을 받아올 수 있습니다. 얻고자 하는 전역변수의 이름(문자열)을 함수의 인자로 넣어주면 해당 전역변수의 값을 반환하거나 변수가 존재하지 않는다면 "undefined" 키워드를 반환합니다.

주의! 만약 전역변수가 존재하지 않는다면 함수는 undefined을 반환할 것입니다. 그리고 만약 존재하지 않는 변수의 값을 함수의 인자로 넣게 된다면 반환값으로 undefined 키워드를 출력할 것입니다. 그리고 이것은 당신의 게임에서 에러를 일으키고 함수가 동작할 때 게임이 멈춰버릴 수 있습니다. 혹시 이런 증상이 의심이 된다면 먼저 variable_global_exists함수를 사용해서 인자로 넣은 변수가 존재하는지 확인하시길 바랍니다.

 

문법:

variable_global_get(name);

인자값(Argument) 설명(Description)
name 얻고 싶은 전역변수의 이름(문자열)

 

반환값:

Value (any data type) or undefined (if the named variable does not exist)

 

Example:

if variable_global_exists("enemy_num")
   {
   show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
   }

위 코드는 전역변수가 존재하는지 여부를 판단하고 존재한다면 그 값을 콘솔창에 출력을 합니다.
