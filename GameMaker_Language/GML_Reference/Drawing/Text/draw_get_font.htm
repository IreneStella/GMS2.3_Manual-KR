draw_get_font

이 함수는 현제 할당된 폰트를 반환합니다. 만약 폰트가 없다면 -1을 반환하거나 할당된 폰트의 ID값(양의 정수)을 얻습니다.

 

문법:

draw_get_font();

 

반환값:

정수 (-1 혹은 폰트의 ID값)

 

예제:

if draw_get_font() != fnt_MainMenu
    {
    draw_set_colour(c_blue);
    draw_set_font(fnt_MainMenu);
    }
draw_text(room_width / 2, 200, "MENU" );

위 예제는 현재 어떤 폰트를 사용하는지 확인합니다. 만약 폰트가 fnt_MainMenu가 아니라면, 문자를 그리기 전에 폰트를 바꾸고 색도 바꿉니다.
