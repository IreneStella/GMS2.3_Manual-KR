draw_get_valign

이 함수는 현제 조절된 수평기준값을 얻습니다, 수평기준값을 밑에 있는 상수 중 하나를 반환합니다.

 

Syntax:

draw_get_valign();

 

Returns:

Constant

상수 조정
fa_top
fa_middle
fa_bottom

 

Example:

if draw_get_valign() != fa_top
    {
    draw_set_valign(fa_top);
    }
draw_text(100, 32, "Score: " + string(score));

이 위의 코드는 조정을 위쪽을 중심으로 하는지 확인합니다, 만약 그렇지 않으면 위쪽을 중심으로 조정되고 몇몇 문자를 그립니다.
