draw_set_halign

이 함수는 수평기준선을 정하는데 사용됩니다. 정확히 말하면 앞으로 사용될 모든 문자 그리기 수평기준선을 인자에 넣은 상수로 바뀝니다. 기본설정값은 fa_left 입니다. 이 앞에 나온 상수들만 사용가능 합니다:

 

상수 조정
fa_left
fa_center
fa_right

 

문법:

draw_set_halign(halign);

인자 설명
halign 수평기준선.

 

반환값:

N/A

 

예시:

draw_set_halign(fa_left);
draw_text(100, 32, "Score: " + string(score));
draw_set_halign(fa_right);
draw_text(room_width - 100, 32, "Health: " + string(health));

이 위의 코드는 같은줄에 두개의 문자열을 그립니다, Score는 왼쪽 중심이며 Health는 오른쪽 중심으로 조정됩니다.
