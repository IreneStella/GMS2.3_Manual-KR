draw_text_transformed_colour

이 함수는 draw_text()함수에 draw_text_transformed() 그리고 draw_text_colour() 를 합친 함수입니다, 문자의 크기와 각도 뿐만 아니라 변화하며 색을 칠해줄수 있도록 합니다, 기본으로 설정된 알파값(투명도)와 색은 무시합니다.

NOTE: HTML5 타겟일때 WebGL 이 불가능하면 변화 블렌딩(Gradient blending)은 불가능 합니다,비록 블렌드 색을 설정할수 있고 처음에 주어진 색으로 폰트가 그려질수 있지만. 이 방법으로 그려지는 블렌딩은 복사된후 캐시에 저장됩니다 그리고나서 필요하면 사용됩니다, 이는 최적화와 거리가 멀며 만약 당신이 다양한 색을 사용하면 게임은 느려집니다. 그러므로 당신은 font_set_cache_size() 을 사용하여 캐시의 크기를 제한을 걸면 괜찮아집니다.

문법:

draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha);

Argument Description
x 그려질 x의 좌표.
y 그려질 y의 좌표.
string 그려질 문자열.
xscale 수평 크기.
yscale 수직 크기.
angle 문자의 각도.
c1 그려질 문자열 좌측상단의 색.
c2 그려질 문자열 우측상단의 색.
c3 그려질 문자열 우측하단의 색
c4 그려질 문자열 좌측하단의 색.
alpha 문자열의 투명도.

 

반환값:

N/A

 

예시:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5);

이 위의 코드는 주어진 문자열을 룸의 중앙에 그립니다, 기존의 크기보다 2배크게 그리며 image_angle값에 따라 각도가 바뀌며, 노란색에서 빨간색으로 바뀌며 칠해지며 투명도 0.5만큼을 얻습니다.
