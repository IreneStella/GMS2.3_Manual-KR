draw_text_transformed

이 함수는 draw_text()함수와 비슷하게 작동하며 수평 혹은 수직의 크기를 바꿀수 있습니다 (효율적으로 축소와 확대가 가능합니다) 그리고 게임메이커 스튜디오2는 특정한 각도로 문자를 돌립니다.(0은 기본값이며 0을 넘으면 문자는 반시계방향으로 움직입니다).

 

문법:

draw_text_transformed(x, y, string, xscale, yscale, angle);

인자 설명
x 그려질 x좌표.
y 그려질 y좌표.
string 그려질 문자열.
xscale 수평 크기(기본값은 1).
yscale 수직 크기(기본값은 1).
angle 문자의 각도.

 

반환값:

N/A

 

예시:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

이 위의 코드는 룸의 중앙에 문자열을 그립니다, 기본보다 2배의 크기로 그려지며 image_angle값만큼 돌아갈수 있습니다.
