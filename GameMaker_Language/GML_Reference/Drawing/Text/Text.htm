Text

문자를 그릴수 있는 수 많은 함수가 있습니다. 알아야 할 점은 몇몇 함수는 문자의 색과 크기를 원레 주어진 값과 다르게 그릴수있습니다, 그러나 HTML5나 특정 기기에서, 이 함수들은 당신의 게임을 느리게 만들거나 보기 안좋게 만들수 있으므로 각 크기의 글꼴을 따로 저장하는것이 더 좋을수 있습니다, 혹은 큰 글꼴을 이용하여 작게 만드는것이 반대로 사용하는 것보다 더 좋습니다. 또한, 만약 코드 여러줄을 하나의 색을 정하고 싶다면 각각을 확장된 함수를 사용하는것 보다 draw_set_colour() 그리고 draw_set_alpha() 함수를 이용하여 draw_text 함수를 사용하는것이 더 좋습니다.

이 함수들은 그려질 문자열들을 조정하는데 사용되는 함수입니다:

 

  • draw_set_font
  • draw_set_halign
  • draw_set_valign
  • draw_get_font
  • draw_get_halign
  • draw_get_valign

 

이 함수들은 화면에 문자를 그릴때 사용됩니다, 이중 몇개는 간단한 조정으로만 그려지며 몇개는 색과 크기를 변경하며 그릴수도 있습니다:

 

  • draw_text
  • draw_text_ext
  • draw_text_colour
  • draw_text_transformed
  • draw_text_ext_colour
  • draw_text_ext_transformed
  • draw_text_transformed_colour
  • draw_text_ext_transformed_colour

 

이 함수는 내장 변수인 highscore을 이용하여 highscore들을 그릴때 사용됩니다:

 

  • draw_highscore
