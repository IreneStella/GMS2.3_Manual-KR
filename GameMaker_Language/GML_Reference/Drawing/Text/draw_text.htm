draw_text

이 함수를 사용하면 룸에 어디에 있든 문자열을 그릴수 있습니다. (만약 실수 숫자를 사용하고 싶다면 string() 함수를 문자에 감싸면 됩니다.) 만약 문자열을 결합하고 싶으면 +를 사용하면 됩니다. (아래 예제 참고) 또한 \n을 문자열안에 넣으면 줄바꿈이 되어 여러 줄을 그릴 수 있게 됩니다. (어떻게 옳바른 형식을 사용하고 어떤 이스케이프 문자를 사용할 수 있는지는 여기를 참고해 주세요.) 문자의 색과 알파값(투명값)은 미리 설정된 draw_set_alpha() 함수와 draw_set_colour() 함수로 결정됩니다.

NOTE: 정확한 문자의 위치는 draw_set_halign() 와 draw_set_valign()에 영향을 받습니다.

 

문법:

draw_text(x, y, string);

인자 설명
x 그려진 문자열의 x 좌표값.
y 그려진 문자열의 y 좌표값.
string 그려질 문자열.

 

반환값:

N/A

 

예시:

draw_text(x, y, "Hello, " + global.Name + "!\nI hope you are well!");

이 위의 코드는 개체의 x/y 에 문자열이 그려집니다, 이는 전역변수 "Name"에 저장된 문자열을 사용하고 2줄로 나눠집니다.
