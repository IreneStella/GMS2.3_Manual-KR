draw_text_ext

이 함수는 draw_text()와 비슷하게 그릴겁니다 다른점은 각각의 문자열줄의 공간을 설정할수 있습니다 이때 한줄의 픽셀길이가 설정한 최대가로길이 (단위는 픽셀)보다 길다면 게임메이커 스튜디오2는 자동으로 가장 가까운 빈 공간으로 줄바꿈을 합니다. (만약 빈 공간이 없다면 문자는 최대가로길이를 뚫고 그려집니다). 이런이유로 모든 문자열의 시작점에 위치한 빈 공간은 이전에 그려진 문자열의 배치구조를 없에버립니다. 또한 중요한점은 sep인자에 -1값을 넣으면 결정된 글꼴의"M"의 높이를 기준으로 작동됩니다.

 

문법:

draw_text_ext(x, y, string, sep, w);

인자 설명
x 그려질 문자열의 x좌표.
y 그려질 문자열의 y좌표.
string 그려질 문자열
sep 각 문자열줄들간의 픽셀간격.
w 줄바꿈의 기준인 최대가로픽셀값.

 

반환값:

N/A

 

Example:

draw_text_ext(100, 50, keyboard_string, 3, 300);

이 위의 코드는 사용자가 키보드에친 문자를 그립니다, 이때 가로가 300픽셀을 넘을때마다 줄바꿈이 됩니다. 이코드는 또한 줄간의 간격은 3픽셀만큼 차지합니다.
