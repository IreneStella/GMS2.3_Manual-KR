draw_text_ext_colour

이 함수는draw_text() 함수에 draw_text_ext() 와 draw_text_colour() 함수를 섞은 버전입니다, 이를 사용하면 색변경과 일정 픽셀을 넘겼을때 줄바꿈을 동시에 사용할수 있습니다.

NOTE: HTML5 타겟일때 WebGL 이 불가능하면 변화 블렌딩(Gradient blending)은 불가능 합니다,비록 블렌드 색을 설정할수 있고 처음에 주어진 색으로 폰트가 그려질수 있지만. 이 방법으로 그려지는 블렌딩은 복사된후 캐시에 저장됩니다 그리고나서 필요하면 사용됩니다, 이는 최적화와 거리가 멀며 만약 당신이 다양한 색을 사용하면 게임은 느려집니다. 그러므로 당신은 font_set_cache_size() 을 사용하여 캐시의 크기를 제한을 걸면 괜찮아집니다.

 

문법:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

인자 설명
x 그려질 문자열의 x좌표.
y 그려질 문자열의 y좌표.
string 그려질 문자열
sep 각 문자열줄들간의 픽셀간격.
w 줄바꿈의 기준인 최대가로픽셀값.
c1 그려질 문자열 좌측상단의 색.
c2 그려질 문자열 우측상단의 색.
c3 그려질 문자열 우측하단의 색
c4 그려질 문자열 좌측하단의 색.
alpha 문자열의 투명도.

 

반환값:

N/A

 

예시:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

이 위의 코드는 유저가 키보드로 입력한 문자열을 그립니다, 만약 문자열의 가로가 300픽셀을 넘기면 줄바꿈을 하며 줄 사이의 간격은 5픽셀만큼 차지합니다. 모든 줄들의 문자는 위쪽은 밝은 파란색, 밑은 어두운 파란색으로 바뀌는색으로 그려집니다.
