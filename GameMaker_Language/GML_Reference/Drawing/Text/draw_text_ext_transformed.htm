draw_text_ext_transformed

이 함수는 draw_text() 함수에 draw_text_ext() 와 draw_text_transformed() 함수를 합친버전 입니다, 이는 각 줄의 최대길이와 빈 공간을 유지시켜주면서 문자의 크기와 회전값을 바꿀수 있도록 해줍니다. 중요한건 인자의"w"은 크기를 1을 기준으로 둡니다, 그래서 만약 크기가 1이 아니라면, 크기값이 바뀐만큼의 비율로 변경됩니다 . 예를들어, 2의 크기로 설정하고 300픽셀마다 줄바꿈을 하라고 인자에 넣으면, 문자는 예상과 다르게 주어진 픽셀을 넘길 것입니다. 대신 크기에 맞춰 150픽셀로 설정하면 잘 작동됩니다.

 

문법:

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle);

인자 설명
x 그려질 문자열의 x좌표.
y 그려질 문자열의 y좌표.
string 그려질 문자열
sep 각 문자열줄들간의 픽셀간격.
w 줄바꿈의 기준인 최대가로픽셀값.
xscale 수평 크기.
yscale 수직 크기.
angle 문자의 각도.

 

반환값:

N/A

 

예시:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle);

이 위의 코드는 룸 중앙에 주어진 문자열을 그립니다, 각 줄의 최대길이는 300픽셀이며, 각 줄간의 공간은 10픽셀 입니다, 기존의 크기보다 2배 크며 image_angle값만큼 돌아갑니다.
