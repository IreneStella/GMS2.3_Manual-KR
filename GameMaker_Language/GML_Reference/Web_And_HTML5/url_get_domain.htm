url_get_domain

이 함수는 HTML5 게임에서 게임이 실행되는 사이트의 주소를 가져옵니다.

 

문법:

url_get_domain();

 

반환값:

문자열 (string)

 

예시:

dom_name = url_get_domain();

이 예시는 게임이 실행되는 페이지의 도메인을 변수 "dom_name"에 저장합니다.
